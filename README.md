# Sasha's Zen

A personal browser-based trainer for preparing for emotionally difficult work conversations.

## What's inside

- **Phrase dictionary** — reframe reactive impulses into calm, professional responses
- **Response cards** — ready-made phrases by situation: victimhood, personal attacks, fact distortion, ultimatums, escalation
- **Simulation** — live role-play with an AI playing the difficult colleague, three difficulty levels, hints and per-reply coaching
- **Affirmations** — guided texts to read aloud before a call (30 sec and 1 min), an in-call emergency anchor, and a post-call recovery mode
- **Breathing** — four breathing patterns with a visual timer: box breathing, 4-7-8, 6-2-8, and a quick desk version
- **Speech analysis** — paste your own words after a call to get a count of filler words, justifications, and confidence phrases — plus AI feedback on tone

## How to use

Open the link in any browser. No installation, no account, no data stored.

On iOS, voice recording is unavailable — use the manual text input instead, analysis works the same way.

## Stack

Single HTML file. Vanilla JS. Claude API for simulation and speech analysis.
